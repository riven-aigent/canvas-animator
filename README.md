# Canvas Animator

A generative art animation tool built with SvelteKit and HTML5 Canvas.

## Features

- **4 Animation Presets**: Particle Flow, Sine Waves, Golden Spiral, Matrix Rain
- **Real-time Rendering**: Smooth 60fps animations using HTML5 Canvas
- **Frame Export**: Download any frame as PNG
- **Responsive Design**: Works on desktop and mobile

## Tech Stack

- Svelte 5
- SvelteKit
- HTML5 Canvas API
- Vite

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
